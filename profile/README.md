# CodeOps Studio

A plugin-powered mobile IDE for Android OS. Code, run and deploy anywhere anytime – without a laptop.

## Our Mission

Build a **decentralized, community-driven IDE** for Android OS.

CodeOps Studio exists because we believe coding should have no limits. Whether you're learning, shipping, or just curious — this platform is for you.

- **For beginners** – Start learning to code today. No laptop needed. Just your phone and curiosity.
- **For enthusiasts** – Your phone is your playground. Build code and excitement.
- **For pros on the go** – Code when inspiration strikes — anywhere, anytime.

## Projects

| Repository | Description |
|------------|-------------|
| [`plugin-sdk`](https://github.com/codeops-studio/plugin-sdk) | The plugin SDK — the foundation to build plugins that run on Android |
| [`plugin-runtime`](https://github.com/codeops-studio/plugin-runtime) | The plugin sandbox that enforces security |
| [`ide`](https://github.com/euptron/CodeOps-Studio) | CodeOps Studio — the IDE itself |
