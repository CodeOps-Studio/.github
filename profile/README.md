# CodeOps Studio

A plugin-powered mobile IDE for Android OS. Code, run and deploy anywhere anytime – without a laptop.

## Our Mission

Build a **decentralized, community-driven IDE** for Android OS.

CodeOps Studio exists because we believe there should be no limit to coding. Whether you're learning, shipping, or just curious — this platform is for you.

- **For beginners** – Start learning to code today. No laptop needed. Just your phone and curiosity.
- **For Enthusiast** – Your phone is your playground. Build code & excitement.
- **For Pros on the go** – Code when the inspiration strikes anywhere anytime.


## Projects

| Repository | Description |
|------------|-------------|
| [`pugin-sdk`](https://github.com/codeops-studio/plugin-sdk) | The plugin SDK – the foundation used build plugins that on Android |
| [`plugin-runtime`](https://github.com/codeops-studio/plugin-runtime) | The plugin sandbox that makes enforces security |
| [`ide`](https://github.com/euptron/CodeOps-Studio) | CodeOps Studio – the IDE itself |
